## Projects

### 01 — NSE Pre-Expiry Effect Analysis

_Does the 2-day window before Thursday F&O expiry create
abnormal returns across NSE sectors?_

Tested across 6 indices · 24 years of data · t-test
statistical validation

**Key finding:** Statistically significant in Nifty IT
only (p=0.02) — driven by high options open interest
in the IT sector.

→ [View project](projects/Thursday-Effect-NSE/)

### 02 — F1 Podium Probability Model

_Using logistic regression on 7 years of race data to
predict podium probability for any driver at any circuit._

Features: qualifying position · recent form · circuit
history · team · season performance rate

**Model accuracy: 90.3%** (5-fold cross validation)
